**Never compromise Priority 1** - get SOMETHING on arXiv by Day 14.

### If Technical Issues:

**LaTeX won't compile:**
- Use Overleaf (cloud-based, always works)
- Have backup on local machine

**Figures not rendering:**
- Export as PDF (vector) or high-res PNG (raster)
- Include as external files, not embedded

**Code won't run:**
- Create minimal working example
- Include in SI as pseudocode if necessary
- Link to GitHub for full version

**GitHub/Zenodo issues:**
- Can submit without DOI initially
- Add DOI in arXiv update (allowed)
